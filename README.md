# Pesquisômetro

- [x] Criação do Repositório

- [ ] Estruturação de JSON com os resultados em si da eleição
- [ ] Frontend para mostrar resultados baseado nos dados
- [ ] Frontend de apresentação do app
- [ ] Colocar em produção
